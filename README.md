# lucide-game-Descubra-o-cone
Um mini-game no estilo lucide.dev para aprender e memorizar ícones da biblioteca Lucide. Feito com React + Tailwind + lucide-react.

Preview React Tailwind License

Demo: jogue direto no preview aqui do chat ou faça deploy na Vercel em 1 clique.

Como jogar
O jogo te desafia: Qual ícone é responsável por [ação]?

São 15 desafios com ícones reais da Lucide: play, pause, volume-2, search, trash-2, settings e mais
3 opções por rodada, só 1 é a correta
Errou? O ícone pisca em vermelho e treme por 400ms
3 erros no mesmo ícone = perde 1 vida e pula automaticamente
Você começa com 3 vidas. Perdeu as 3 = Game Over
Acertou = +1 ponto e animação verde
Regra simples inspirada no seu pedido original: lógica de pisca e lógica de vida.

Stack
React 18 + Vite
Tailwind CSS para o layout minimalista no estilo lucide.dev
lucide-react para os ícones
100% client-side, sem backend
